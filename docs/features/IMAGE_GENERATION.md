# Image Generation

CompareIntel supports comparing AI image generation models side-by-side. Users can enter a prompt and see images generated by multiple models simultaneously.

## Model Discovery

Image-capable models are identified via OpenRouter's `output_modalities` field. Use `GET https://openrouter.ai/api/v1/models?output_modalities=image` to discover available models.

## Admin Flow for Image Models

When adding a model via the admin panel:

1. **Detection:** The add flow fetches OpenRouter model data and checks `output_modalities` for `"image"`.
2. **Registry:** Image models are added with `supports_image_generation: true` and `category: "Image"`.
3. **Skip scripts:** Do **not** run `setup_model_renderer.py` (no math/formatting analysis) or `research_model_benchmarks.py` (Help Me Choose is text-focused).

## Credit Rules

- **Token-based models** (e.g., Gemini): Use existing `calculate_credits()` when `usage` is present.
- **Per-image models:** Use `credits = pricing.image × CREDITS_PER_DOLLAR × num_images` from OpenRouter metadata.
- **Fallback:** If `pricing.image` is missing, use `IMAGE_CREDITS_PER_GENERATION`.

Constants in `backend/app/config/constants.py`:
- `CREDITS_PER_DOLLAR` - Converts OpenRouter pricing.image ($) to credits
- `IMAGE_CREDITS_PER_GENERATION` - Fallback when pricing unavailable

## Tier Limits

- `FREE_TIER_IMAGE_COMPARISONS_PER_DAY` - Free tier daily image comparison limit
- `UNREGISTERED_IMAGE_COMPARISONS` - Unregistered users cannot use image models (0)

## Unregistered User Gating

Unregistered users see image models in the list but cannot select them. Clicking an image model shows a sign-up modal. The Compare button is effectively disabled (no models selected).

## Aspect Ratio and Resolution

In image mode, Advanced settings show aspect ratio (e.g. 1:1, 16:9) and image size (1K, 2K, 4K). Each model has tested capabilities stored in the registry (`image_aspect_ratios`, `image_sizes`). Incompatible selections trigger a conflict modal and block the change. See `backend/scripts/test_image_config_aspect_ratio.py` and `backend/data/image_config_test_results.json`.
