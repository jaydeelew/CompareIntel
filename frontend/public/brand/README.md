# Brand Icons

All icon assets for CompareIntel. Generated by `npm run generate:icons` (see `scripts/generate-icons.js`).

## Inventory

| File                                           | Size      | Purpose                                                                                                   |
| ---------------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------- |
| `tab.svg`                                      | vector    | Browser tab favicon (high-contrast)                                                                       |
| `CI_favicon_tab_blue.svg`                      | vector    | Alternate tab favicon (blue); may be used for browser tab                                                 |
| `CI_favicon_tab_white.svg`                     | vector    | Alternate tab favicon (white); may be used for browser tab                                                |
| `screenshot-desktop.png`                       | 1280×720  | PWA manifest screenshot (wide)                                                                            |
| `screenshot-mobile.png`                        | 601×1334  | PWA manifest screenshot (narrow)                                                                          |
| `logo.svg`                                     | vector    | In-app UI (nav, hero, offline); apple-touch-icon SVG                                                      |
| `favicon.ico`                                  | 16+32+48  | Crawlers; legacy browsers; SERP favicon (also copied to `public/favicon.ico` for `/favicon.ico` requests) |
| `icon-16.png`                                  | 16×16     | Optional `rel="icon"`                                                                                     |
| `icon-32.png`                                  | 32×32     | Optional `rel="icon"`                                                                                     |
| `apple-touch-icon-180.png`                     | 180×180   | iOS / iPadOS home screen                                                                                  |
| `pwa-any-192.png`                              | 192×192   | PWA manifest `purpose: any`; shortcuts; `msapplication-TileImage`                                         |
| `pwa-any-512.png`                              | 512×512   | PWA manifest `purpose: any`; JSON-LD logo; in-app branding                                                |
| `pwa-any-1024.png`                             | 1024×1024 | PWA manifest `purpose: any`; shortcuts; Windows install / taskbar                                         |
| `serp-circle-192.png`                          | 192×192   | SERP / circle-safe variant                                                                                |
| `serp-circle-512.png`                          | 512×512   | CTA; offline fallback; SERP when must match ICO                                                           |
| `pwa-maskable-48.png` … `pwa-maskable-512.png` | 48–512    | PWA `purpose: maskable` (Android adaptive icons)                                                          |

## Sources

- `tab.svg` ← `CI_favicon_tab.svg`
- `logo.svg` ← `CI_Icon.svg`
- PWA any PNGs ← `CI_favicon_1024x1024.png` (full-bleed) or `CI_Icon.svg`
- SERP / favicon.ico ← `CI_favicon_circle_safe.svg`
- Maskable ← `CI_Icon.svg` with 20% padding
- icon-16, icon-32, apple-touch-icon-180 ← `CI_favicon_tab.svg`
